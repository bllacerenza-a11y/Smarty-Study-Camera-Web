
# Smarty Study Camera

Projeto desenvolvido para o FIAP Challenge em parceria com a JOVI pelo grupo FutureCall.

O Smarty Study Camera é uma solução criada para transformar a câmera do smartphone em uma ferramenta inteligente de aprendizagem, utilizando conceitos de OCR, inteligência artificial, automação e experiência do usuário para auxiliar estudantes no registro, organização e revisão de conteúdos acadêmicos.

---

# Sobre o Projeto

Muitos estudantes utilizam diariamente a câmera do celular para fotografar:
- lousas
- slides
- códigos
- anotações
- exercícios
- resumos

Porém, esse processo normalmente gera problemas como:
- imagens tortas
- baixa legibilidade
- reflexos
- desorganização
- dificuldade para localizar conteúdos posteriormente
- perda de produtividade durante revisões

O Smarty Study Camera foi idealizado para solucionar esses problemas por meio de uma experiência mais inteligente, automatizada e organizada.

---

# Objetivo da Solução

O principal objetivo do projeto é utilizar a câmera do smartphone de forma diferenciada para melhorar a experiência de estudos dos usuários.

A proposta consiste em criar um sistema capaz de:
- identificar automaticamente ambientes de aula
- otimizar capturas de conteúdos acadêmicos
- organizar informações automaticamente
- gerar apoio de estudos utilizando inteligência artificial

# Considerações Finais

O Smarty Study Camera busca demonstrar como tecnologias como visão computacional, OCR e inteligência artificial podem transformar a câmera do smartphone em uma ferramenta inteligente de apoio aos estudos.

O projeto une conceitos de:
- experiência do usuário
- front-end
- interatividade
- automação
- organização acadêmica

com foco em melhorar a produtividade e a experiência de aprendizagem dos estudantes.
